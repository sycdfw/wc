秦曼云瞪大着双眼，紧紧盯着孟君良。

　　佛门佛法东渡，找人向西求取真经，而孟君良受到高人指点，同样是一路向西，不同的是，一个是向西求取真经，一个是向西传道！
　　佛门是为了将佛法传于世间。
　　那高人为什么要向西传道，到底有什么深意？
　　而当书生讲到收服孙悟空，让其保护唐僧去西天取经时，秦曼云更是呼吸粗重起来。
　　孙悟空桀骜不驯，本领强大，而且被佛门镇压了五百年，如何会心甘情愿的保护唐三藏去西天取经？
　　孟君良看了秦曼云一眼，今天却是讲得比往常要多。
　　一直讲到孙悟空戴上紧箍方才停止。
　　“行者见说得有理，真个不敢动身，只得回心，跪下哀告道：“师父！这是他奈何我的法儿，教我随你西去。我也不去惹他，你也莫当常言，只管念诵。我愿保你，再无退悔之意了。”三藏道：“既如此，伏侍我上马去也。”
　　那行者才死心塌地，抖擞精神，束一束绵布直裰，扣背马匹，收拾行李，奔西而进。毕竟这一去，后面又有甚话说，且听下回分解——”

　　她的脸色不断的变化，时而惊怒，时而绝望，俏脸一片煞白。
　　她悟了！
　　高人故事里的暗示她悟了！
　　只是，知晓了真相的她却感觉更加的无力。
　　都说大佬们喜欢以天地为棋子，玩弄苍生命运，看来果然如此。
　　如来让人西天取经，不就是在布局吗？所谓的唐三藏和孙悟空不过是他安排的棋子。

　　天地不仁以万物为刍狗，我们又该何去何从？

　　“仙凡之路断绝？”洛诗雨微微一愣。
